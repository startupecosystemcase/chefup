# Инструкция по настройке изображений

## Получение прямых ссылок из Google Диска

1. Откройте файл на Google Диске
2. Нажмите правой кнопкой мыши → "Получить ссылку" или "Поделиться"
3. Установите доступ "Все, у кого есть ссылка"
4. Скопируйте ссылку (формат: `https://drive.google.com/file/d/FILE_ID/view?usp=sharing`)
5. Извлеките `FILE_ID` (часть между `/d/` и `/view`)
6. Используйте формат: `https://drive.google.com/uc?export=view&id=FILE_ID`

## Альтернативный способ (рекомендуется для продакшена)

1. Скачайте все изображения из Google Диска
2. Загрузите их в папку `public/`:
   - `public/cities/` - фотографии городов
   - `public/team/` - фотографии команды
   - `public/partners/` - логотипы партнеров
   - `public/hq.jpg` - фото штаб-квартиры
3. Обновите пути в `lib/images.ts` на относительные (например, `/cities/astana.jpg`)

## Обновление путей

Откройте файл `lib/images.ts` и замените `YOUR_FILE_ID_*` на реальные ID файлов из Google Диска.
