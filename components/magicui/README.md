# Magic UI Components

Magic UI интегрирован в проект. Компоненты можно добавлять через:

```bash
npx shadcn@latest add [component-name]
```

Или вручную копируя компоненты с сайта [magicui.design](https://magicui.design)

## Доступные компоненты

- Animated Beams
- Animated Gradient Text
- Animated Shiny Text
- Border Beam
- Bento Grid
- Globe
- Grid Pattern
- Particles
- Shiny Button
- Shimmer Button
- Sparkles
- Text Reveal Card
- И многие другие...

## Использование

```tsx
import { ComponentName } from '@/components/magicui/component-name'
```
